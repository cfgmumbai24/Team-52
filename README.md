# Team-52 Clueless Coders <?/>


## Installation

Client Side: -
```bash
  cd client
  npm install
```

Server Sider
```bash
    cd server
    npm install
    touch .env
```



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file (In Server Folder)

`MONGODB_URL` 

`JWT_SECRET` - a random long string
